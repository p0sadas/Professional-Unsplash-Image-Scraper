# Contributing to Unsplash Image Scraper

Thank you for your interest in contributing to the Unsplash Image Scraper! We welcome contributions from the community.

### Reporting Bugs

If you find a bug, please create an issue with:

- A clear, descriptive title
- Steps to reproduce the issue
- Expected behavior vs actual behavior
- Your environment (Python version, OS, etc.)
- Screenshots if applicable

### Suggesting Enhancements

We love to hear new ideas! When suggesting an enhancement:

- Use a clear, descriptive title
- Provide a detailed description of the proposed feature
- Explain why this enhancement would be useful
- Include code examples if applicable

### Pull Requests

1. **Fork the repository** and create your branch from `main`

   ```bash
   git checkout -b feature/amazing-feature
   ```

2. **Make your changes**
   - Follow PEP 8 style guidelines
   - Add docstrings to all functions and classes
   - Include type hints where appropriate
   - Write clear, descriptive commit messages

3. **Test your changes**
   - Ensure the code runs without errors
   - Test with different search queries and image counts
   - Verify error handling works properly

4. **Update documentation**
   - Update README.md if you've added new features
   - Update docstrings and comments
   - Add examples if relevant

5. **Submit your pull request**
   - Provide a clear description of the changes
   - Reference any related issues
   - Wait for review and address any feedback
